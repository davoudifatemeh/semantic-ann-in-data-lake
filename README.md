# Semantic Annotations on Joinable Columns Detection in Data Lakes
 ## Repository structure
 `SchemaExtraction/` – utilities/scripts for extracting statistics and sample values from data-lake tables for each column.

`SemanticAnnotation/` – semantic annotation pipeline (assigning new meaningful names (semantic annotations) to columns).

`SemLink/` – SemLink approach source code (semantic linking/matching logic).

`DeepJoin/` – DeepJoin algorithm source code.
